# Data Engineering Using SQL

This folder focuses on **data engineering concepts implemented using SQL**.
It covers data modeling, schema design, incremental loads, SCD, and performance optimization.

## Topics Covered
- Table & schema design
- Star schema modeling
- Incremental data loads
- Slowly Changing Dimensions (Type 2)
- Partitioning strategy
- Indexing
- Data quality checks

These concepts are used in real-world data warehouses (Snowflake, BigQuery, Redshift).
